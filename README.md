# InfraMap

self hostowana strona internetowa do tworzenia mapy infrastruktury serwerów.

UI:
coś ala edytowalny canvas gdzie możemy dodać ikonkę serwera / pc / routhera / switcha itp itd
przypisać im adresy prywatne i publiczne IP, opisy
umieścić w sieci np LAN / innej sieci / w chmurze

układ ui, ułożenie, config itp będą zapisywane plikach json

stack: 
backend - Go lang
frontend - html, js, css
